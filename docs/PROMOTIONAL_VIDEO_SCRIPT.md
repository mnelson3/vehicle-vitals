## Production Notes

### Visual Style

- **Color Palette:** Use brand colors from Vehicle Vitals (check branding guide if available)
- **Screenshots:** Use actual images from docs/screenshots/ folder
- **Animations:** Smooth transitions between screenshots, subtle zoom/pan effects
- **Text Overlays:** Clean, modern sans-serif font (e.g., Inter, Montserrat)

### Audio

- **Voice-Over:** Professional, conversational tone — not overly salesy
- **Music:** Modern tech/productivity vibe (royalty-free options: Epidemic Sound, Artlist, AudioJungle)
- **Sound Effects:** Subtle notification sounds, button clicks (optional)
- **Pacing:** Slightly slower for clarity; allow 0.5s pauses between major thought transitions

### Platform-Specific Recommendations

**YouTube (16:9):**

- Use full video length (3:20)
- Add YouTube cards / end screens for CTA
- Optimize thumbnail: Show clean vehicle image + bold text "Vehicle Vitals"

**TikTok / Instagram Reels / YouTube Shorts (9:16):**

- Use 60-second version
- Large, easy-to-read text overlays
- Vertical framing of screenshots (may require cropping / reframing)

**LinkedIn (1:1 or 16:9):**

- Use 2-minute version
- Add professional testimonials (optional)
- Tone: B2B focus on fleet management, cost savings

**Email Campaign:**

- Embed 60-second version in email
- Add clickable CTA button

## Animation / Interactive Elements (Optional Enhancements)

- **Callout arrows** pointing to key UI elements
- **Number counters** animating on cost figures (e.g., "$644" counts up from 0)
- **Fade-in effects** for lists (7 record categories, filter buttons)
- **Highlight boxes** that zoom/expand around important features
- **Smooth screen transitions** between pages (not hard cuts)

## Accessibility Notes

- Add captions/subtitles for all dialogue (required for social media auto-play)
- Ensure sufficient color contrast for all text overlays
- Describe visual elements in captions for visually impaired viewers
- Test on various screen sizes and devices before publishing
