## 🎯 How to Use These Assets

### Scenario 1: Creating a YouTube Video

1. Read [PROMOTIONAL_VIDEO_SCRIPT.md](PROMOTIONAL_VIDEO_SCRIPT.md) — Full storyboard with scene descriptions
2. Use the full 3:20 version as your guide
3. Reference the actual screenshots from `docs/screenshots/`
4. Create video following the scene breakdown
5. Add captions from script text
6. Optimize title/description with SEO keywords (in script)

**Time to produce:** 2–4 hours (with existing assets)

### Scenario 2: Recording a Podcast Episode

1. Read [PROMOTIONAL_PODCAST_SCRIPT.md](PROMOTIONAL_PODCAST_SCRIPT.md) — Full 8-minute script
2. Record with professional voice talent or host
3. Add background music (recommendations in script)
4. Edit and add intro/outro
5. Upload to podcast platforms
6. Create clips for social media

**Time to produce:** 3–5 hours (with voice talent provided)

### Scenario 3: Running a Social Media Campaign

1. Use 60-second video version for TikTok, Instagram Reels, YouTube Shorts
2. Pair with social copy from the script
3. Create 5–10 variations (different angles: problem, solution, testimonial, feature)
4. Post daily to TikTok, 3–4x per week to Instagram/YouTube
5. Use as ads on Facebook, TikTok Ads

**Time to produce:** 4–6 hours for initial creation + 1 hour daily for posting

### Scenario 4: Email Campaign

1. Start with email sequences in [MARKETING_STRATEGY.md](MARKETING_STRATEGY.md#5-email-marketing)
2. Customize subject lines and copy for your audience
3. Include embedded video or link to video
4. Add CTAs to start free trial
5. Track opens, clicks, conversions

**Time to produce:** 2–3 hours to customize templates

### Scenario 5: Paid Advertising

1. Use 30-second video for YouTube pre-roll ads
2. Use 60-second video + ad copy for Facebook / Instagram ads
3. Use carousel ads with screenshots for product showcase
4. Reference audience / messaging in [MARKETING_STRATEGY.md](MARKETING_STRATEGY.md#4-paid-advertising)
5. Run A/B tests on copy, creative, audiences

**Time to produce:** 3–4 hours to set up campaigns

## 💰 Production Requirements & Costs

### High-Quality Video Production

- **Professional voiceover:** $100–300
- **Background music:** $20–100 (royalty-free licensing)
- **Video editing:** $500–2,000 (outsourced) or ~40 hours if in-house
- **Graphics/animations:** $200–500
- **Total:** $800–2,900

### Podcast Production

- **Professional voiceover:** $100–300
- **Background music/sound:** $20–100
- **Audio editing:** $200–500 (outsourced) or ~10 hours if in-house
- **Podcast distribution:** Free (use Anchor, Buzzsprout, etc.)
- **Total:** $320–900

### Social Media + Content

- **Content creation:** 80–120 hours (internal)
- **Design templates:** $100–200
- **Stock footage (optional):** $50–200
- **Total:** $150–400 + time

### Paid Advertising

- **Recommended budget:** $1,500–3,000/month
- **Expected trial signups:** 30–100/month
- **Cost per trial:** $15–50
